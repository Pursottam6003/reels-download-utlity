# Reels Downloader Frontend (MVP)

Simple React + Vite frontend for the Reels Downloader backend. Features:
- Navbar
- Search bar to paste a video URL
- Displays metadata and available formats with download/stream buttons

Run locally:

```powershell
cd frontend
npm install
npm run dev
```

The frontend expects the backend at `http://127.0.0.1:8000` by default (see `src/App.jsx`).
